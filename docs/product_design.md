# 产品设计文档

## 1. 产品定位

面向 A 股 500 支核心股票池的个人量化研究平台。

## 2. 目标用户

- 个人量化研究者
- 量化入门学习者
- 希望用数据辅助决策的个人投资者

## 3. 第一阶段目标

- 建立可运行、可扩展的工程骨架
- 固化研究平台的目录结构、配置方式和存储方案
- 为后续股票池、历史数据、因子和回测模块预留接口

## 4. MVP 定义

V0.1 的 MVP 是一个能初始化环境、创建数据库结构、展示基础 UI、并可通过测试的项目骨架。

## 5. 产品边界

- 只面向 A 股研究
- 当前只覆盖核心股票池场景
- 当前只做项目骨架，不做真实业务流程

## 6. 不做什么

- 不做实盘交易
- 不做自动下单
- 不接券商 API
- 不提供投资建议

## 7. 核心业务流程

股票池管理 -> 数据初始化 -> 数据更新 -> 数据质量检查 -> 因子计算 -> 评分排序 -> 回测分析 -> 报告输出
